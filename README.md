
# Advent of Code Solutions 🎄

This repository contains my personal solutions for the [Advent of Code](https://adventofcode.com/) annual programming challenges. Advent of Code is an Advent calendar of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language.

## 📂 Project Structure

The project is organized by year, with each day having its own directory containing the puzzle input and the solution scripts.

```text
.
├── 2023
│   └── day_6
│       ├── input.txt   # Puzzle input
│       ├── main.py     # Part 1 solution
│       └── main2.py    # Part 2 solution
├── 2024                # Upcoming/Ongoing challenges
├── 2025                # Future challenges
└── LICENSE
```


## 🚀 How to Run

To run a specific solution (e.g., 2023 Day 6), navigate to the folder and run the Python script:

**Bash**

```
cd 2023/day_6
python main.py
```
